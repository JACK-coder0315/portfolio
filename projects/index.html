MCM 2025 Essay: Navigating the Future: Transformative Traffic Solution for Baltimore

Innovations

  • Established multiple mathematical models (such as the Bridge-Collapsed Effect Model, Bus Stop Optimization Model, Kruskal-Edmonds Traffic Revitalization Model) to comprehensively optimize the transportation network in Baltimore.
  • Utilized various algorithms (such as Dijkstra's algorithm, Particle Swarm Optimization algorithm, Kruskal's algorithm, Edmonds-Karp algorithm) to simulate and optimize traffic flow.
  • Proposed a bus stop optimization strategy based on community division to improve the efficiency and fairness of the public transportation system.

Key Methodologies

  • Data Preprocessing: Removed redundant data and visualized key data for better analysis.
  • Model I: Bridge-Collapsed Effect Model
    • Used the BPR algorithm to simulate traffic flow and determine the relationship between road capacity and width.
    • Used Dijkstra's algorithm to simulate the shortest travel time before and after the bridge collapse.
  • Model II: Bus Stop Optimization Model
    • Used the Haversine formula to convert latitude and longitude to distance.
    • Used Dijkstra's algorithm to simulate the shortest paths between current transportation nodes.
    • Used the Particle Swarm Optimization (PSO) algorithm to simulate the geographical location of new bus stops.
    • Used the K-mean clustering method to divide Baltimore into three neighborhoods and control the variance of bus stop distances within each neighborhood.
  • Model III: Kruskal-Edmonds Traffic Revitalization Model
    • Used Kruskal's algorithm to optimize the connectivity between traffic nodes.
    • Used Edmonds-Karp's algorithm to optimize the distribution of traffic on each road.

MCM 2024 Essay: How On Earth Does The Momentum Control The Match?

Innovations

  • Proposed using the Entropy Weighting Method (EWM) and Analytic Hierarchy Process (AHP) to quantify momentum in tennis matches and established a correlation model between momentum fluctuations and player success.
  • Used the K-means clustering method to divide momentum into five phases and used Mahalanobis distance to predict momentum fluctuations in real-time during matches.
  • Provided strategic advice to players based on momentum fluctuations to improve their performance.

Key Methodologies

  • Problem 1: Determine which player performs better at a given time
    • Used EWM to assign weights to holding serve and breaking serve.
    • Established an average point differential model to evaluate player performance.
  • Problem 2: Investigate whether momentum fluctuations are related to player success
    • Quantified success based on players' cumulative points.
    • Used AHP and consistency checks to determine momentum quantification model weights.
    • Used Python to fit the quantified success model and momentum model.
  • Problem 3: Determine the most significant factors influencing momentum fluctuations and predict momentum fluctuations in matches
    • Used EWM to calculate weights.
    • Identified the most relevant factors for momentum fluctuations (such as ACEs, untouchable shots, break points).
    • Used the K-means clustering method to group sample data into five momentum phases.
    • Used Mahalanobis distance to predict momentum fluctuations during matches.
  • Problem 4: Test the predictive ability of the established model and evaluate its generality
    • Input the test dataset into the K-means clustering model.
    • Used 0-1 normalization to standardize the data.
    • Evaluated the model using binary classification model evaluation methods.
